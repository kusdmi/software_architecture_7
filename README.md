# Микросервисы аутентификации и публикации сообщений

Данный проект представляет собой два микросервиса на **Python + FastAPI**, работающих в контейнерах Docker под управлением **Docker Compose**. Первый сервис занимается регистрацией и аутентификацией пользователей с выдачей JWT-токенов, второй – принимает сообщения от авторизованных пользователей и сохраняет их в базу данных.

## Архитектура

- **auth_service** – сервис аутентификации (порт 8001)
  - Регистрация новых пользователей (пароль хэшируется bcrypt)
  - Аутентификация и выдача JWT-токенов (время жизни 60 минут)
- **post_service** – сервис постов (порт 8002)
  - Приём сообщений только с валидным JWT-токеном в заголовке `Authorization`
  - Сохранение сообщений в базу данных (таблица `messages`)
- **PostgreSQL** – общая база данных (порт 5432)
  - База `auth_db` – таблица `users` (id, email, password)
  - База `post_db` – таблица `messages` (id, user_id, time, message)

## Технологии

- Python 3.9
- FastAPI
- SQLAlchemy
- PostgreSQL 13
- JWT (python-jose)
- bcrypt (passlib)
- Docker, Docker Compose

## Требования

- Установленные **Docker** и **Docker Compose**
- Свободные порты: 5432 (PostgreSQL), 8001 (auth), 8002 (post)
